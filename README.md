✨ Aura Testimonials
Aura Testimonials is a premium, high-fidelity testimonial card generator designed for creators, entrepreneurs, and developers. It allows users to build stunning, professional-grade social proof cards and export them as ultra-sharp PNG images in seconds.

🚀 Features
📸 Ultra-HD Export: Powered by html2canvas at 5x scale for retina-level clarity (no more blurry text).

🖼️ Anti-Stretch Technology: Smart frame cropping ensures profile photos never look squashed or stretched, regardless of original aspect ratio.

🎨 Dynamic Theme Engine: Real-time color pickers to customize Accent and Border colors to match any brand identity.

📱 Fully Responsive: A modern CSS Grid layout that adapts seamlessly from desktop to mobile.

🗑️ Dynamic Management: Add unlimited cards and remove specific ones with a single click.

✨ Premium Aesthetics: Deep rounded corners, smooth transitions, and a dark-mode-first design.

🛠️ Tech Stack
HTML5: Structured semantic markup.

CSS3: Custom properties (Variables), Grid, and Flexbox for a robust layout.

JavaScript: Vanilla JS for DOM manipulation and file handling.

Library: html2canvas for high-resolution rendering.

📖 How to Use
Configure: Enter the client's name, role, and their glowing testimonial.

Upload: Choose a profile image (the system will automatically center-crop it).

Customize: Use the "Accent" and "Border" color pickers to style the card.

Generate: Click Generate Card to see it appear in the workspace.

Export: Click Export All PNG to download high-resolution versions of your cards.
